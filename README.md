# Markdown Previewer
A custom Markdown previewer and editor built from scratch in Python and Tkinter

## Compatibility and Architecture
- **OS:** macOS
- **Architecture:** Apple Silicon
- **Build Format:** macOS Application Bundle
  
## Features
- Dual panel layout (Editor + Preview)
- Real time Markdown rendering (`# Headers`, `**Bold**`, `- Lists`)
- Dynamic doc stats (line, word, characters counter)
- Dark/light theme toggle
- `.md` file saving

# How to run and play
1. Download `Markdown_Previewer.zip` from Github Releases and extract it
2. **First run:** Open terminal in the extracted folder and clear the macOS quarantine flag:
   ```
   xattr -cr Markdown_Previewer.app
   ```
3. Use the tool :)
